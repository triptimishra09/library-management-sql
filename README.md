# library-management-sql
Library Management System using MySQL

##  Project Overview

This project is a **Library Management System database** built using **MySQL**.
It demonstrates how relational databases can be used to manage books, authors, members, and borrowing records in a library.

##  Technologies Used

* MySQL
* SQL
* GitHub for version control

##  Database Structure

The system contains the following tables:

1. **Authors** – stores information about book authors
2. **Books** – stores book details and availability
3. **Members** – stores library member information
4. **Borrow_Records** – tracks which member borrowed which book

##  Database Concepts Used

* Primary Keys
* Foreign Keys
* Table Relationships
* SQL Queries

##  Project Files

* `schema.sql` – database schema and table creation queries
* `sample_data.sql` – sample data for testing
* `queries.sql` – useful SQL queries for data retrieval
